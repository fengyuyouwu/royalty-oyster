# 皇家蚝 Royalty Oyster

🎓 **广州大学 机械与电气工程学院** — 全自动生蚝开壳取肉机团队

融合机械结构设计、嵌入式控制、计算机视觉与物联网技术的工程创新团队。

## 项目简介

全自动生蚝开壳取肉机通过**限位扩缝、水平翻转、红外推送、夹持开壳、翻转取肉**五大机构与视觉/电控系统协同，实现从入料到取肉弃壳的全流程自动化作业。

## 技术栈

- 🔧 **机械结构**: SolidWorks · ANSYS 有限元 · 斜面分型虎钳 · 四杆+杠杆 10:1 力放大
- ⚡ **嵌入式控制**: STM32F407VET6 · Keil5 · 自研四层PCB · PID 闭环控制
- 👁️ **视觉与感知**: YOLOv8-seg · OpenCV · 光谱分析 · 数字孪生 NX MCD
- 🌐 **物联网**: Modbus-RTU · WiFi/蓝牙 · 端云协同四层架构
